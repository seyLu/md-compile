# md-compile
Compile multiple markdowns into one README

### Requirements
`python>=3.11.3`

`node>=18.15.0`

### Install dependencies and build
- NPM

    ```shell
    npm install
    npm run build
    ```

- Virtual Env

    ```shell
    pip install -r requirements.txt
    ```

### Running on local

```shell
flask run
```

### Developing on local

```shell
npm run watch
```
